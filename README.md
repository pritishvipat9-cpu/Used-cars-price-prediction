# Used Car Price Prediction

## Project Overview
This project predicts the market value of used vehicles based on vehicle specifications and historical data.

## Business Problem
Accurate vehicle price estimation helps buyers and sellers make informed decisions in the used car market.

## Skills Demonstrated
- Data Cleaning
- Feature Engineering
- Regression Modeling
- Data Visualization

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib

## Models Used
- Linear Regression
- Random Forest Regressor
- XGBoost Regressor

## Evaluation Metrics
- MAE
- RMSE
- R² Score

## Results
Built regression models capable of estimating used vehicle prices with strong predictive performance.

## Future Improvements
- Additional vehicle features
- Advanced ensemble models
- Real-time prediction system
